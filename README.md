draft-dodger
===========

Temporarily add jekyll post drafts to your gitignore

[![NPM](https://nodei.co/npm/draft-dodger.png?downloads=true)](https://nodei.co/npm/draft-dodger/)

![](http://i.imgur.com/anZ7GCd.png)

* `npm install -g draft-dodger`
* `draftdodger`
